## Getting Started

### Prerequisites

Use the package manager [npm](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm) to install project

### Installation

1. Clone the repo

```bash
    git clone https://github.com/NhatTuanNguyen/shop.git
```

2. Install NPM packages

-   API:

```bash
    cd shop & npm install
```
3. Running

```
bash npm start
```
